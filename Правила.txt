Правила для Германии: За победу в матче команда получает 3 очка, ничья отмечается одним очком, поражение баллов не приносит. (До сезона-1995/96 победитель получал 2 очка за игру.) В случае равенства очков у двух и более команд учитываются дополнительные показатели: разница забитых и пропущенных мячей, общее количество забитых голов во всех играх. Если эти значения также равны, проводится сравнение результатов тех матчей, которые команды сыграли друг с другом, затем сравнивают разницу мячей между этими клубами, если этот показатель равный, тогда смотрят количество забитых мячей на выезде между этими клубами, затем смотрят общее количество забитых мячей в лиге на выезде(последние два параметра опущены для соответсвия правилам франции). Если после всех этих сравнений определить команду, которая должна стоять выше, не удается, тогда назначается дополнительный матч «плей-офф»(так же опущен по условию задания)
Правила для Франции: Итоговое положение команд в турнирной таблице зависит от количества набранных очков. Дополнительными параметрами являются разница голов и количество забитых мячей. В случае если данные компоненты также совпадают, тогда берется во внимание количество набранных очков в матчах друг с другом, затем разница мячей в этих играх
Правила для Италии: Победа в матче приносит клубу 3 очка, ничья – одно, за поражение команда не получает баллов. В тех случаях, если две или более команд получают одинаковое количество очков, регламент чемпионата предусматривает дополнительные показатели. Наиболее приоритетным из них (после количества очков) является количество набранных пунктов в играх между этими командами. Если и этот показатель оказывается равным, тогда во внимание берутся следующие показатели:

Разница забитых и пропущенных голов в матчах между собой
Количество забитых мячей в матчах между собой
Общая разница голов во всех матчах
Количество забитых голов во всех матчах
Количество голов забитых на выезде во всех матчах
